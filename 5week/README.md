### 기본과제
### **`STEP09 - Concurrency`**

- 본인의 애플리케이션 내에서 발생할 수 있는 동시성 이슈를 식별
- 동시성 이슈를 해결하기 위해 적합한 DB Lock 적용 및 AS-IS / TO-BE 비교 및 보고서 작성

> 보고서는 `문제 식별` - `분석` - `해결` - `대안` 의 항목들을 기재해 주시기 바랍니다.
>
